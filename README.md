# HandBrasil Stats App

Aplicacao Flutter com Supabase para administracao, scout ao vivo e visualizacao de partidas.

## Deploy web

O projeto ja esta preparado para subir no Vercel.

Guia completo:

- [DEPLOY_VERCEL.md](./DEPLOY_VERCEL.md)

## Variaveis de ambiente

Crie um arquivo `.env` local com base no `.env.example`.

Use apenas a chave publica do Supabase no app web:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
